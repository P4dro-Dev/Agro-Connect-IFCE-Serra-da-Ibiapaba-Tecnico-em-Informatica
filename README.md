# Agro-Connect-IFCE-Serra-da-Ibiapaba-Tecnico-em-Informatica
O AgroConecta Ibiapaba é uma solução tecnológica desenvolvida no âmbito do IFCE, resultante da sinergia. A plataforma entrega valor imediato ao produtor através de cotações em tempo real e um balcão de negócios digital, fortalecendo a economia local através da tecnologia.
